# Sunways-NT-Inverter-ESPHome
An ESPHome-based data logger for Sunways NT series solar inverters. Data is collected via RS232 or RS485 (with daisy-chain support) and seamlessly integrated into Home Assistant for monitoring and visualization.


## Contribution
Feel free to contribute to this project! You can assist me by testing the code / hardware setup on your devices, reporting bugs, improving documentation, adding new features, or submitting pull requests. Thanks in advance!
